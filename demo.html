Dúvidas Frequentes

Encontre respostas para as perguntas mais comuns sobre o FluxoApp

O FluxoApp é gratuito?

Sim, o FluxoApp possui uma versão gratuita com recursos básicos para controle financeiro. Oferecemos também um plano Premium com funcionalidades adicionais por uma assinatura mensal.

Meus dados estão seguros?

Sim, levamos a segurança dos seus dados muito a sério. Todas as informações são criptografadas e protegidas com os mais altos padrões de segurança do mercado.

Posso usar o FluxoApp sem conexão com a internet?

Sim, o aplicativo móvel funciona offline. Você pode adicionar despesas e receitas mesmo sem conexão, e os dados serão sincronizados quando você estiver online novamente.

Como faço para exportar meus dados?

Na versão web, acesse Configurações > Exportar Dados para baixar um arquivo CSV com todas as suas transações. Na versão móvel, essa opção está disponível no menu de configurações.
